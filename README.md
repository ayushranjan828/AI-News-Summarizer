# 📰 AI News Summarizer

An AI-powered News Summarizer built using Streamlit, LangChain, Tavily Search API, and Mistral AI. This application fetches the latest news based on a user query and generates concise bullet-point summaries using Large Language Models (LLMs).

## 🚀 Features

- 🔍 Fetches real-time news from the web
- 🤖 AI-generated news summaries
- 📌 Clean bullet-point output
- ⚡ Fast and interactive Streamlit UI
- 🌐 User can search any topic dynamically

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- Tavily Search API
- Mistral AI
- dotenv

## 📂 Project Workflow

1. User enters a topic/query
2. Tavily Search fetches latest news articles
3. LangChain processes the data
4. Mistral AI summarizes the news
5. Streamlit displays summarized bullet points
